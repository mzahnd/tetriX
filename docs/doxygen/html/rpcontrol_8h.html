int rpi ( void  )

Run game on Raspberry PI.

The game runs on a display and it is controlled by a Rpi.

Parameters
None.
Returns
NO_ERROR==0
ERROR=-1

Create words for the tetris menu.

Array of pointer to numbers.

Creates the gameboard

Variable that takes joystick state.

Variable that gives difficulty mode

An array of pointer to words with a counter to show different menu options in the display. The ones that are in the same "page" are separated from the others with a NULL pointer.

Initializes the display, audio, joystick and gameboard.

Clears the display.

Starts the tetris music if everything went OK.

Starts the introduction animation.

Until the joyswitch is pressed, it doesn't show the menu.

It shows the menu starting with PLAY.

Until the joyswitch is unpressed, it doesn't get into the menu.

The menu control.

It analyses the joystick position.

If the user goes right or left, it increase or decrease the counter that goes threw the array of words.

If the joyswitch is pressed it analyses which one is the option chosen.

If it is play, it opens the play menu.

If it is sound, it goes to On/Off.

If it is on, it plays the music.

If it is off, it pauses the music.

If it is exit, it finishes the loop.

If it is mode, it goes to Easy/Hard.

If it is easy or hard, it puts the game on that mode.

After the counter change, it analyzes where is it on the array. If it is pointing a NULL, it goes back(or forward) so as to comeback to the last word.

It clears the menu but not the TeTrIs sign.

It prints the word chosen, if it is too long, it prints words that move from right to left.

Unless the joystick goes to the center it doesn't continue. This is done so as to make the user go to right or left every time he wants to change the menu option.

It stops the music. User chose EXIT to finish the program.

It inicialyzes and plays a goodbye song.

It makes an animation saying goodbye(same as hello twice) THIS IS HERE BECAUSE WE ARE HAVING TROUBLE WITH THE MUSIC.

It turns off the display.
