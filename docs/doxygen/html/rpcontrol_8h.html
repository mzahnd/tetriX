The menu, it includes pause menu.

It analyzes the joystick position.

If the user goes right or left, it increase or decrease the counter that goes threw the array of words.

If the joyswitch is pressed it analyzes which one is the option chosen.

If it is play, it opens the play menu.

It doesn't start until joyswitch is unpressed

If it is the first time playing, it starts the stats, timer and gameboard.

It comes with the instruction of pausing the game or ending it.

If it is sound, it goes to On/Off.

If it is on, it plays the music.

If it is off, it pauses the music.

If it is exit, it finishes the loop.

If it is mode, it goes to Easy/Hard.

If it is easy or hard, it puts the game on that mode.

After the counter change, it analyzes where is it on the array. If it is pointing a NULL, it goes back(or forward) so as to comeback to the last word.

It clears the menu but not the TeTrIs sign.

It prints the word chosen, if it is too long, it prints words that move from right to left.

It prints tetrix at the top.

Unless the joystick goes to the center it doesn't continue. This is done so as to make the user go to right or left every time he wants to change the menu option.
