Clear a square in the display.

It clears a part of the display chosen by the user. It makes a square (widthXheight) on the coordinate the user gives it(the origin is on the top left corner).

Parameters
Square'swidth.
Square'sheight.
Coordinate'sX value.
Coordinate'sY value.
Returns
Nothing

It goes threw cols&rows turning off the leds until it reaches the end of the square or the border of the display.

Y value will be the origin plus the actual row.

X value will be the origin plus the actual column.

It turns off the point.
