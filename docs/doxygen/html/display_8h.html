Detailed Description

Visual module to show on the display the gameboard, scores and menu.

It has all the matrices of letters and numbers used for showing the user the menu and scores. It also has animations and functions to show the gameboard.

Print words with an animation.

It prints words(made by letters) on the display with a space between them. It is similar to printW but this one can show bigger words displacing the letters from right to left.

Parameters
Pointerto letters_t.
Coordinate'sX value.

Returns

X is the coordinate of the last letter of the word.

Next is a flag to know if the last coordinate x of the last letter has an acceptable displacement.

Jmove takes surf() value(so as to avoid calling the function so many times)

It doesn't continue until the joystick is in the center(as in rpcontrol)

It clears the screen.

It prints the word and it make a pause then.

It makes a little pause until it starts moving.

Unless the user moves the joystick to the left, right or press the switch button, it show the full word going from one side to the other.

It clears the display every time there is a new printW.

It gets the displacement after printing the word.(If it is a long, word, X will be huge)

If it is not as displaced to the left as it should It fix origin one place to the left.

If it is totally displaced to the left, it starts displacing the word to the right until it is totally displaced.

This flag blocks the first condition so as to displace the word totally to the right and then start again.

If it was totally displaced to the right, it starts again (to the left).
