◆ surf()

int surf ( void  )

Analyze the joystick status.

It gives the user the Joystick and Joyswitch status.

Parameters
None.
Returns
Joystick or Joyswitch status.

Initializes a variable and a structure with the value of the joystick coordinates and if the switch was pressed or not.

Depending on the joystick and joyswitch position it returns with a result showing the state.

If the switch is pressed.

If the X coordinate too big(to the right)

If the X coordinate too small(to the left)

If the Y coordinate too big(to the top)

If the Y coordinate is too small(to the bottom)

If it is not in any directions.

It shouldn't happen because joystick will be always in one direction or another but jut for precaution there is NOTHING state.
