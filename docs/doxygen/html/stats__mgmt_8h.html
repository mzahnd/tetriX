typedef struct STATS stats_t

Holds current game statistics.

Read and write the current game stats in this structure. Includes the current level, how many lines have been cleared, the next piece, how many pieces of each kind have been deployed and the top and actual scores.

Note
For "next" use PIECE_ definitions in board.h
