Detailed Description

Read and write scores in and from the top score list.

Gives information about the top scores, as well as a NAMESIZE-1 long string with the name of the author and allows adding a new score to the list.

Note
Trying to add a new score that is lower than the lowest score already in the list will do nothing. This is made for always calling the function after a game has ended.
Warning
Function init_rwOps() must be called before accessing any field in this structure.
