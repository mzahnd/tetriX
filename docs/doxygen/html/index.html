tetriX documentation

Introduction

tetriX is a Tetris game written in C, meant to be runned on any Linux distribution or in a Raspberry Pi with a matrix display + joystick module attached.

Source code organization

The project source code is located inside the src folder and it is clearly devided in backend and frontend related sources.
Also, the whole backend is tested using CUnit test framework and its tests are located in tests folder, which is in the project parent directory.

Note
Altough all tests are properly documented, that documentation is not located here but inside each test.
This is meant to keep the documentation more clear and show the programmer only the important stuff.
