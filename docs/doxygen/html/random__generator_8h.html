Detailed Description

Functions used to generate a bag of random pieces.

Generates an array (called "bag of pieces") of size NUM_PIECES (defined in @header board.h) with random pieces following the set of rules taken from tetris.wiki webpage:

  • No more than 4 S or Z pieces togheter.
  • No more than 12 pieces between two different I pieces.

Generates a NUM_PIECES size bag with a pseudo-random list of pieces.
